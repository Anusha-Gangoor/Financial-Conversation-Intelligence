# Financial Conversation Intelligence System

This project focuses on converting real-world financial conversations into structured insights using AI.

## Features
- Speech-to-text conversion
- Financial entity detection (EMI, loans, SIP)
- Structured summary generation
- Multilingual and code-mixed language support

## Tech Stack
- Python
- NLP (Natural Language Processing)
- Speech Recognition (Whisper / Indic ASR)

## Description
This system captures everyday financial discussions and transforms them into actionable insights to help users make smarter financial decisions.
